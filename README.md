\# Cybersecurity Awareness Chatbot (Emmie)



\## Overview

This project is a C# console-based cybersecurity awareness chatbot named \*\*Emmie\*\*.  

The chatbot helps users learn about important cybersecurity topics such as passwords, phishing, and safe browsing.



The program was developed as part of the \*\*PROG6221 Programming POE assignment\*\*.



\---



\## Features

\- ASCII banner interface

\- Personalized greeting using the user's name

\- Typing effect for chatbot responses

\- Cybersecurity topic menu

\- Keyword recognition (user can type numbers or words)

\- Random cybersecurity tips

\- Welcome audio playback

\- Colored console output

\- Structured code using classes

\- Continuous Integration using \*\*GitHub Actions\*\*



\---



\## Cybersecurity Topics Covered

\- Password Safety

\- Phishing Attacks

\- Safe Browsing

\- General cybersecurity awareness tips



\---



\## Technologies Used

\- C#

\- .NET Console Application

\- GitHub Version Control

\- GitHub Actions (CI)



\---



\## How to Run the Program

1\. Clone the repository.

2\. Open the project in \*\*Visual Studio\*\*.

3\. Build and run the application.

4\. Enter your name and interact with Emmie using the menu.



\---



\## Continuous Integration

This repository uses \*\*GitHub Actions\*\* to automatically build the project when changes are pushed.



A successful CI run confirms that the project compiles without errors.
